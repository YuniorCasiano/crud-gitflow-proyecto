# Sistema de Gestión de Tareas - CRUD con Git Flow

Proyecto CRUD completo implementando la metodología Git Flow correctamente.

## Estructura del Proyecto
```
crud-gitflow-proyecto/
├── index.html
├── styles.css
├── script.js
└── README.md
```

## Características del Proyecto

- **CRUD de Usuarios**: Crear, leer, actualizar y eliminar usuarios
- **CRUD de Tareas**: Gestión completa de tareas con prioridades
- **CRUD de Proyectos**: Administrar proyectos con fechas
- **Diseño minimalista**: Fondo blanco, alineación a la izquierda

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
- Git Flow

## Git Flow Implementado

Este proyecto sigue la metodología Git Flow con:
- Rama main (producción)
- Rama developer (desarrollo)
- Rama qa (testing)
- 5 ramas feature/hotfix

## Cómo usar

1. Abrir `index.html` en un navegador
2. Gestionar usuarios, tareas y proyectos
3. Todos los datos se almacenan en memoria
